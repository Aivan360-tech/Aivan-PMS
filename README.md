# Parking Management System

A complete React frontend for an enterprise Parking Management System.

## Stack

- React 19
- Vite
- Tailwind CSS
- React Router DOM
- React Icons
- Local JSON data

## Pages

- Login
- Dashboard
- Parking Slot Master

## Run

```bash
npm install
npm run dev
```

The app runs at the local URL shown by Vite, usually `http://127.0.0.1:5173/`.

## Data

Parking records are stored in `src/data/parkingData.json`. Dashboard metrics and parking slot cards are generated from this file.
